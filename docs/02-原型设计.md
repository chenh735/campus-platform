# 校园课程学习与项目协作平台 - 原型设计

## 1. 原型说明

本原型使用 Markdown 文本线框图描述页面结构和操作流程。界面风格以简单实用为主，优先使用列表、表格、表单、按钮和详情面板，不追求复杂视觉效果。

## 2. 用户端整体导航

用户端顶部导航：

```text
┌──────────────────────────────────────────────────────────────┐
│ 软件工程学院学习协作平台        课程    项目招募    个人中心 │
│                                            登录/注册 或 用户名 │
└──────────────────────────────────────────────────────────────┘
```

主要页面：

- 登录页
- 注册页
- 课程列表页
- 课程详情页
- 项目招募列表页
- 发布项目招募页
- 个人中心页

## 3. 登录页

```text
┌──────────────────────────────┐
│ 登录                         │
├──────────────────────────────┤
│ 邮箱：[                    ] │
│ 密码：[                    ] │
│                              │
│ [登录]                       │
│                              │
│ 没有账号？去注册             │
└──────────────────────────────┘
```

交互流程：

1. 用户输入邮箱和密码。
2. 前端校验不能为空。
3. 后端校验账号密码。
4. 登录成功后进入课程列表页。
5. 登录失败时提示错误原因。

## 16. 主要操作流程

### 16.1 注册登录流程

```text
进入注册页
  -> 输入 mail2.sysu.edu.cn 邮箱
  -> 发送验证码
  -> 填写验证码、昵称、密码
  -> 注册成功
  -> 登录
  -> 进入课程列表页
```

### 16.2 课程评价流程

```text
课程列表
  -> 查看课程详情
  -> 点击我要评价
  -> 填写评分和评价
  -> 提交
  -> 课程平均评分更新
  -> 评价出现在评价列表
```

### 16.3 上传资料流程

```text
课程详情
  -> 学习资料 Tab
  -> 上传资料
  -> 填写标题、标签、简介并选择文件
  -> 提交审核
  -> 管理员审核通过
  -> 资料可下载
```

### 16.4 点赞获得经验流程

```text
用户B 查看用户A的资料或评价
  -> 点击点赞
  -> 点赞记录写入数据库
  -> 用户A经验值 +1
  -> 用户A等级自动重新计算
```

### 16.5 发布项目招募流程

```text
项目招募页
  -> 点击发布招募
  -> 填写项目信息
  -> 发布
  -> 招募出现在项目列表
  -> 其他用户申请加入
  -> 发布者在个人中心查看申请
```

### 16.6 管理员添加课程流程

```text
管理员登录
  -> 课程管理
  -> 新增课程
  -> 填写课程代码、名称、教师、学分、类别、简介
  -> 保存
  -> 用户端课程列表展示新课程
```
